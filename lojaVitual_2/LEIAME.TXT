INSTALAÇÃO - versão de testes
-----------------------------

Esta é a quinta versão do phpwebthings, e por enquanto
a instalação é feita manualmente através da edição de
alguns arquivos. Eu espero que as próximas versões
contenham algumas configurações que possam ser feitas
através do browser.

Muitas alterações foram feitas até o momento desde a versão
0.4, e não será muito fácil converter da 0.4 para essa (0.5).
Mas não é impossível! ;) Eu terei que fazer isso quando for
atualizar o phpdbform.com e phpdbform.com.br.

Como essa versão ainda não está pronta, o portal está somente
em inglês, desculpem-me, mas assim que os arquivos de 
internacionalização estiverem mais maduros, a próxima língua
será o Português!

Esta versão tem muitas novidades:
- Novidades: Agora são separadas por categoria, e será fácil
criar links apenas para as categorias;
- Usuários: Novo cadastro com muito mais campos, possibilidade
de visualizar um "cartão" de cada usuário. Livro de endereços
muito mais completo, e futuramente mais estatísticas, como 
número de novidades, faq e outras enviadas;
- Side boxes: agora é possível incluir arquivos PHP dentro 
das caixas laterais!
- Modular: Agora é fácil escolher quais módulos você deseja
utilizar, e mais fácil ainda criar novos!
- Downloads: Descrições, miniaturas, screenshots, muito melhor
que antes;
- Fórum: Michel Weimerskirch está desenvolvendo um novo fórum!
- PostgreSQL: o phpwebthings irá rodar nele também!
- Pic of the Day: seleciona uma figura para mostrar por um dia,
também mostra um álbum com todas as imagens, além de comentários
para cada uma das imagens;
- FAQ: estão de volta;
- Banners: idem;

Agora, o fórum ainda não está funcionando completamente (está sendo
refeito), mas logo, logo ele estará de volta.
Provavelmente após esta versão irei lançar o release estável
com correções de bugs e com o fórum pronto.


Como instalar:

Primeiro des-zipe ou des-tarreie (aff, acabei de inventar)
o arquivo para um diretório de sua escolha, crie um banco
de dados no seu mysql e execute o script webthings_mysql.php nele.
Edite o arquivo wt_config.php :

Escolha a língua que deseja utilizar no phpwebthings no
começo do arquivo, deixando somente uma delas sem comentários
(o padrão é Inglês).

Title é o título do seu site, que irá aparecer como o título
das janelas e em alguns e-mails enviados pelo aplicativo. O
site_url é muito importante, pois através dele o phpwebthings
irá montar o link para o usuário ativar a sua conta. Name_admin
e email_admins são utilizados pelos e-mail enviados para o
usuário.

A configuração do banco de dados:
"host" = ip ou domínio onde o banco de dados está instalado, se
   for na mesma máquina que oservidor web, deve-se utilizar
   "localhost";

"user" = usuário com permissão de acesso ao banco de dados do
   phpwebthings (insert,delete,update);

"password" = senha do usuário acima;

"database" = nome do banco de dados com as tabelas do phpwebthings.


Você pode escolher o tema, simple ou black ou criar o seu próprio
tema! Dê uma olhada na pasta themes para aprender como são feitos
os temas!
* O tema deafult agora é o "nt", mas o tema simple ainda existe!

O tema templ01 é utilizado pelo phpdbform e só o altere se desejar
alterar a aparência dos formulários de administração do site.

A parete do menu no arquivo config.php lista todos os links que
irão aparecer na caixa principal na esquerda.

Agora execute o seu site. Já existe um usuário cadastrado no
phpwebthings, é o "admin" com a senha "123", por favor altere
esta senha assim que possível! É simples, faça o login e vá
para a "Minha Conta" e altere a senha. No users admin você
pode alterar o nome do usuário também.

As classes de usuário são:

0 - Normal User : todo mundo
1 - post_news : O usuário pode enviar novidades para o site sem
    necessitar da aprovação do moderador;
2 - post_faq : O usuário poderá adicionar faqs sem necessitar da
    aprovação do moderador;
4 - banner : O usuário pode editar os banners;
8 - superuser : O usuário pode fazer tudo, até excluir você do
    site! (cuidado)

Você pode somar os valores para misturar os direitos (o superusuário
já tem todos os direitos)
Por exemplo, um usuário pode enviar novidades e faqs sem necessitar
 do moderador, portanto sua classe é 3.

Quando um usuário tem permissão para fazer algo, uma caixa de
administração irá aparecer na esquerda (admin box)

Todos os textos utilizados no site estão guardados no diretório
"lang", você pode editá-los para usar os textos que quiser, só
tenha cuidado com os %s e %d utilizados nos textos.
