INSTALL
-----------------------------

Это пятый релиз phpwebthings и на данный момент для установки
требуется редактирование файлов вручную. Надеюсь, что в следующих
версиях большую часть процесса установки можно будет провести через
браузер.

По сравнению с версией 0.4, были сделаны некоторые изменения и потому
процесс апгрейда нетривиален, но вполне реален :)
По крайней мере, сайты phpdbform.com, phpdbform.com.br и некоторые другие
были обновлены без особых проблем или потери данных.

В этой версии добавлены следующие функции:
- Категории новостей: Все новости теперь разбиты по категориям, потому
сортировка стала гораздо более удобной;
- Регистрация: В форме, заполняемой при регистрации, были добавлены новые поля. 
При нажатии на имя пользователя выводится аккуратная "визитная карточка". Улучшена 
Адресная книга, в дальнейшем туда будут добавлены элементы статистики: количество 
новостей, вопросов FAQ и т.п., отправленых пользователем
- Боковые колонки: Теперь в них можно вставлять PHP код! Круто, не правда ли? :))
- Разделение на модули: Каждая функция интерфейса (кроме логина) вынесена в отдельный 
модуль - выбирайте те, что вам нужны, а также создавайте собственные - теперь это
делается легче, чем когда-либо!
- Файлы: К каждому файлу можно добавить короткое и длинное описание, картинки, скриншоты...
- Форум: Michel Weimerskirch разрабатывает новый форум, гораздо лучше прежнего!
- PostgreSQL: теперь можно использовать не только MySQL, но и PgSQL!
- Фото дня: эта система каждый день случайным образом выбирает одну картинку из фотоальбома
и выводит на главную страницу. Кроме того, можно просматривать весь фотоальбом, с thumbnails и 
комментариями для каждой фотографии;
- ЧаВО: снова работает;
- Banners: idem;

Форум пока не готов. Вскоре он будет включен в основную версию. После тест-релиза будет 
очередной релиз стабильной версии с исправленными ошибками и работающим форумом.


Установка:
-------------

Разархивируйте скачанный .tar.gz или .zip файл в любую директорию по выбору,
создайте новую базу данных на сервере и выполните скрипт webthings_mysql.php.
Измените wt_config.php в соответствии с вашими предпочтениями:

Выберите язык интерфейса в начале файла. Для этого раскомментируйте нужную 
строку, но после этого убедитесь, что остальные остались закомментированными.
По умолчанию используется английский язык.

Title - название сайта, эта строка будет выводиться в заголовке окна браузера
и при составлении некоторых Email-сообщений.

Site URL - очень важный параметр, именно этот адрес указывается в письме, которое
получает посетитель при регистрирации для подтверждения.

Name_admin и email_admin - указываются в письмах пользователям.

Данные о базе данных должны соответствовать той конфигурации, которые вы используете
на своем сервере. Указанный пользователь должен иметь достаточные полномочия для 
добавления, удаления и изменения таблиц phpwebthings.

Вы можете выбрать тему оформления из тех, что имеются в папке themes, а также 
создать свою собственную.

Тема templ01 является стандартной темой phpdbform, внесенные изменения скажутся только
на внешнем виде страниц системы администрирования.

Раздел menu файла wt_config.php содержит ссылки, которые будут отображаться в главном меню,
которое располагается в левом верхнем углу страницы.

После сохранения настроек можно проверить, что получилось в результате.
ВНИМАНИЕ! В базе данных пользователей, которая создается при установке, уже есть один 
login: admin, pass: 123 - измените его как можно раньше! Для смены пароля воспользуйтесь
страницей "Настройки", а для изменения логина выберите на панели управления пункт 
"Пользователи"

Используемые классы пользователей:

0 - Обычный посетитель (по умолчанию)
1 - post_news : Пользователь может публиковать новости без обязательной проверки их модератором
2 - post_faq : Тоже самое, но для вопросов/ответов ЧаВО
4 - banner : Пользователь может изменять список баннеров
8 - superuser : Суперюзер может все!

При сложении этих значений получается пользователь, который может выполнять несколько функций 
сразу (суперюзер имеет все полномочия)
К примеру, пользователь, который может публиковать новости и добавлять записи ЧаВО будет иметь
класс 3

На панель управления выносятся только те функции, которые доступны конкретному пользователю.
Обычные пользователи не имеют панели управления.

Весь текст интерфейса находится в файлах, расположенных в папке lang (некоторые модули имеют свои
собтвенные папки lang). Если вы хотите изменить какую-то строку - изменяйте соответствующий файл,
только не забывайте про специальные символы %s и %d, которые там иногда встречаются.

phpwebthings и phpdbform могут содержать ошибки и недоработки - если вы найдете что-то подобное,
то не забудьте сообщить об этом мне.
